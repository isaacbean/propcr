 Connect to Wifi Network: OFF
This setting will connect the proPCR to a separate wifi network from its own (2.4GHz networks only). After entering credentials, you will need to restart the proPCR to initiate the connection. Upon successful connection, the proPCR will have a unique IP address so that it can be accessed and programmed from the separate wifi network. The proPCR will continue to broadcast its own wifi network "proPCR.", which is where the unique IP address will be displayed.
On startup proPCR will connect to the network: 
Wifi Network Name: The name (SSID) of the network you would like the proPCR to join when it restarts next time.
Wifi Password: (Optional) The password of the network you would like the proPCR to join.

 Broadcast proPCR Network: ON
This setting turns on and off the wifi network that the proPCR broadcasts: "proPCR.". You must be connected to another wifi network to turn the broadcast off. If you encounter difficulties connecting to the proPCR after disabling the broadcast, then you can always manually reset your proPCR by double pressing the reset button (refer to the documentation on propcr.com for more information on manually resetting).

You will need to restart the proPCR before the changes take effect. It may be helpful to 'forget' the network in your network preferences before rejoining with the new password.

The Ramp Function: This tab is where you can create, save, and run ramp programs. There are no cycles in a ramp program. It creates a linear change of the block temperature over a set period of time. A ramp is useful for things like controlled annealing of oligos.

Gradient Ramp Off

Start Temp: The starting temperature of the ramp.

End Temp: The ending temperature of the ramp.

Low Start Temp: The low gradient starting temperature of the ramp.

High Start Temp: The high gradient starting temperature of the ramp.

Low End Temp: The low gradient ending temperature of the ramp.

High End Temp: The high gradient ending temperature of the ramp.

Ramp Time: The amount of time for the ramp.
The Heat Block Function: This tab is where you can create, save, and run heat block programs. There are no cycles in a heat block program. It holds the block at a steady temperature for a set period of time. A heat block program is useful for things like running isothermal reactions.

Gradient Heat Off

Heat Block Temp: The temperature of the block during the program.

Low Heat Block Temp: The low gradient temperature of the block during the program.

High Heat Block Temp: The high gradient temperature of the block during the program.

Heat Block Time: The amount of time for the program to run.

Initial Denature Temp: The initial denature happens once at the start of a program to separate double stranded DNA into single stranded DNA. 95° celcius is a standard temperature for denaturing DNA.

Initial Denature Time: 2-5 minutes (120-300 seconds) are common times for the initial denaturation. More time may be needed if using a "hot start" polymerase.

Touchdown (TD) Phase

Touchdown PCR: More information about Touchdown PCR avaliable at: https://genome.cshlp.org/content/4/5/S185.full.pdf+html

TD Number of Cycles: The number of touchdown cycles in the touchdown phase.

TD Denature Temp: The temperature for each denature step in the touchdown phase. 95° celcius is common.

TD Denature Time: The time for each denature step in the touchdown phase. 30 seconds is common.

TD Start Anneal Temp: The temperature of the first anneal step in the touchdown phase. 10° higher than the calculated anneal temperature is common.

TD End Anneal Temp: The temperature of the last anneal step in the touchdown phase. 1° above the desired anneal temperature is common (either at the calculated temperature or 5° below it).

Low TD Start Anneal Temp: The low gradient temperature of the first anneal step in the touchdown phase. 10° higher than the calculated anneal temperature is common.

High TD Start Anneal Temp: The high gradient temperature of the first anneal step in the touchdown phase.

Low TD End Anneal Temp: The low gradient temperature of the last anneal step in the touchdown phase. 1° above the desired anneal temperature is common (either at the calculated temperature or 5° below it).

High TD End Anneal Temp: The high gradient temperature of the last anneal step in the touchdown phase. 1° above the desired anneal temperature is common (either at the calculated temperature or 5° below it).

TD Anneal Time: The time for each anneal step in the touchdown phase. 45 seconds is common.

TD Extend Temp: The temperature for each extend step in the touchdown phase. 72° celcius is common.

TD Extend Time: The time for each extend step in the touchdown phase. 60 seconds per 1000 basepairs to be amplified is common, though some polymerases require more time.

Regular Phase

Number of Cycles: The number of times your program will run through the denature, anneal, and extend steps. 20-30 cycles is common.

Denature Temp: The temperature for each denature step. 95° celcius is common.

Denature Time: The time for each denature step. 30 seconds is common.

Anneal Temp: The temperature for each anneal step. 5° celcius lower than the calculated primer temperature is common.

Low and High temperatures can have a 12 degree range

Low Anneal Temp: The low gradient temperature for each anneal step.

High Anneal Temp: The high gradient temperature for each anneal step.

Anneal Time: The time for each anneal step. 45 seconds is common.

Extend Temp: The temperature for each extend step. 72° celcius is common.

Extend Time: The time for each extend step. 60 seconds per 1000 basepairs to be amplified is common, though some polymerases require more time.

Final Extend Temp: The temperature for the final extend step. 72° celcius is common.

Final Extend Time: The time for the final extend step. 5 minutes (300 seconds) is common.

Lid Temp: A heated lid prevents condensation from forming at the top of the tubes, which would alter the reaction conditions. A few degrees above the highest reaction temperature is common. If you leave this field blank, the lid will not heat. CAUTION! Metal elements may be hot when you open the lid

Program Name: The name of the program if you would like to save it.
